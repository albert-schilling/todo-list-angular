# Angular Task List

Built according to the tutorial by [Unleashed Design](https://www.youtube.com/watch?v=yS6QKQkfw_w&list=PLwG0hu96Rc0oK1MJ3zfFRPqpmMJJ7Bn_g) with Angular, Dragula and JSON Server.

## Useage

1. Clone the repo on your local machine with `git clone https://github.com/albert-schilling/todo-list-angular.git`
2. Requirement: JSON Server. Install globally with `npm install -g json-server`
3. Install packages with `npm run install:all` while inside project folder.
4. To start both server and client concurrently, type `npm start` while inside project folder.
5. Enjoy ‾\\\_(^o^)\_/‾
